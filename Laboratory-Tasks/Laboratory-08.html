TASK L8.T1

Description of the task to be performed:

Practice INSERTING and DELETING some values inside the array:
Write the program, which first creates an array of N numbers (with some predefined content),
and than allows the user to ADD new numbers (at the specified position),
or DELETE numbers which match some user defined criterion (the value or alternatively the position).

The best approach is to create the program MENU, which controls executions of four functions:
  a) void loadValuesIntoArray( int A[ ], int max_capacity, int* used_size );
  b) void printArrayContent( int A[ ], int used_size );
  c)    int insertValueIntoArray( int A[ ], int max_capacity, int* used_size, int value, int position );
  d)    int deleteAllNegativeNumbers( int A[ ], int* used_size );

The problem of inserting and deleting elements in the array, is well explained in material: Data Structures and Algorithms - Array Algorithms

Another collection of advantageous materials (with many example solutions in: C, C++, Java, Python, C#, PHP) can be found at GeeksForGeeks tutorial: Array Data Structure

TASK L8.T2

Description of the task:

SORTING algorithms:
Write your own implementation of "Selection Sort" or "Insertion Sort" algorithm:
   void mySort( int A[], int size )  
to order the contents of the array int A[N] in ascending order.

Add the menu, which allow to test algorithm correctness, using the array initialised with:
1) predefined values:
       int A[N]={ 1, 4, -2, 3, 0, 9 };  
2) or alternatively,
    on a random sequence of values, generated by using the function rand()
