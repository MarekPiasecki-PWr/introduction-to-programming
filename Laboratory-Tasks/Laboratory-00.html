TASK L0.T2

Description of the task:

Write a program that displays (on screen) the form of your "business card"
ie. first/last name, address, phone, e-mail, etc.

------------------------------------
NAME: Jan Kowalski
ADDRESS: Wroclaw Janiszewskiego 11/17
PHONE: 333-444-555
EMAIL: 123456@student.pwr.edu.pl
------------------------------------

Please save your program in the file with meaningful-informative name.
In place of default - very anonymous name "main.c"
the name like "123456_lab5_task3.c" will be better to identify,
that this is a program for task 3, created by the student with ID 123456, at the time of laboratory number 5.


TASK L0.T3

Description of the task:

Extend the first program (from TASK L0.2) in this way, that all personal data (e.g. name, phone, address, ... ) are collected from the keyboard at the program startup.
Finally, the correct "Business card" should appear on the screen, filled with the data just collected.

Hint:  study the W3schools tutorial "C User Input"
