TASK L5.T1

Description of the task to be performed:

Write a program that allows for storing a collection of real numbers (loaded from the keyboard) in the C-array composed of N cells.
The maximal capacity (physical size) of the array, should be defined as const int c(for example N=10).

When entering the data, the program should display user-friendly prompts in the form:
    Tab[1] = 12.0
    Tab[2] = -1.0
    Tab[3] = .......

After completing the input loop, the program should execute following operations:
  1. display all the loaded data ( starting from array[0] to array[n-1] )
  2. display the array content in reverse order (starting from the last element and ending with the first).
  3. calculate the mean (arithmetic average value) from all numbers in array.

TASK L5.T2

Description of the task:

Write a program that fills 10-elemental array with random integers from range [-20,+50],
and then:

a) counts total number of negative and positive numbers in an array
b) calculate extreme values (minimum and maximum)
c) find the second largest element in an array,
d) (*) calculate Mean Squared Error (population variance)


Hint:
The rand() function, from library <stdlib.h>, can be used to generate random numbers.

Functions  srand() and time()  
(the second is from library <time.h>),
can be used to initialise random pseudo-generator.

* As possible extension (improvement), the currently used number of elements (logical size), could be different (smaller) than N , and represented by additional int variable n.
In this case, there are to sizes:
1) physical/maximal constant size N=10
2) logical/used size n  ( variable loaded from keyboard, which should be n<=N )
