TASK L4.T2

Description of the task:

Write a program that loads several numbers from the keyboard,
asking in the meantime, the question "Do you want to finish: Y / N ?"
At the end of the loop, program should display calculated values of:
the arithmetic average and minimum of given numbers.

TASK L4.T3

Description of the task:

Write a program, executing a loop, which is reading integer numbers from the keyboard, until at least one of the below listed conditions is fulfilled:
- The multiplication of these numbers exceeds 1000,
- The number of reported "even" numbers exceed 10,
- Three subsequent numbers will have exactly the same value (e.g.  3, 4, 2, 5, 5, 5 )
