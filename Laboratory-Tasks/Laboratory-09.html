TASK L9.T1

Description of the task to be performed:

1.A) First, please  analyse  standard null-terminated text processing functions from <string.h> library.
Pay special attention to functions from the categories:
  • string manipulation  (e.g. strcpy, strcat )
  • string examination (e.g. strlen, strcmp, strchr, strstr, strtok )
Example programs, illustrating how to use existing functions can be found  here

1.B) Then,  create and test  your own implementations: my_strlen, my_strchr, of standard strlen and strchr functions.
Examples and hints, how create custom/user defined functions, can be found here

TASK L9.T2

Description of the task:

Composing advanced user-defined functions for character string processing (for individual, more unique cases).

a) Review the concept and make sure, that you have a good understanding of how the predefined functions: strcmp and strncmp work.

b) Create your own implementation of the new/unique function:
    int strccmp(const char* text1, const char* text2, char delimiter )
which compares two input texts (text1 and text2) until null-termination '\0' or a specified "delimiter" character.

Example call:
    int result = strccmp( "Abc=def" , "Abc=123" , '=' );
should result with returning a zero value (which means "zero/no difference" )

TASK *L9.EXTRA (optional)

Description of the task:

Implement and test, the text processing function (analogous to strupr/strlwr)
   char* strcpt( char* text );
which converts the input text into "capitalized"  (see: Wiki: Capitalization).
Depending on the language, the rules for capitalization are very complicated.
In this task, the most simplified rule can be used, that every word should start with uppercase,
and all subsequent letters should be lowercase.

Example: "wElcoME in WROclaw!"
should be converted into: "Welcome In Wroclaw!"
