TASK L12.T1

Description of the task to be performed:

In-memory database, using the data representation in the form of array of struct.

Define the N-elements array (pseudo-dynamic with with an additional counter that indicates the size/amount of entered data, or dynamicaly allocated) of structures defined in Task2 from laboratory 11.

Write a program that implements menu with following functionalities of database:
  1. function that reads / adds a new elements to this array,
  2. function that displays all the content of database (all elements of array),
  3. function that modify / updates the content of selected database element,

  4. function that delete / remove the selected element from database,
  5. function that that allows filtering / searching the database to display only elements matching the given criteria.
    For example: display the list of second-hand cars in database, but only "Fiat" brand, not older than 15 years, with vehicle mileage less then 150 thousand kilometers.

TASK *L12.EXTRA (optional)

Description of the task:

Expand the program Lab12_Task1 with additional functionality of alphabetical sorting the database content using two different selected fields/columns as ordering criterion.

For example:
 a) list of cars, alphabetically sorted by brandname  ( Alpha Romeo, Audi, BMW, Citroen, . . . )
 b) list of cars, sorted by price of the car  ( 5400, 7210, 10500, 13000, . . . )
 c) list of cars, sorted by engine type  ( Diesel, Electric, Hybrid, Hydrogen, LPG, Petrol, )
