TASK L2.T1

Description of the task to be performed:

Write a program, which loads (from the keyboard) variables of four basic types:

  1. integer number (preferably short)
  2. floating point number (preferably double)
  3. single character (char)
  4. text (using array text representation char text[30] )
  5. boolean (logical TRUE/FALSE) (int ?)
and then display the collected values, in five following rows, in a formated form:
    • on 15 positions,
    • aligned / justified to right,
    • with maximally 2 fraction digits
Documentation:
  • function scanf 
  • function printf

TASK L2.T2

Description of the task:

Write a program that reads two calendar dates from the keyboard
(each date encoded by three integers: day, month, year)
and checks whether the second date is earlier than the first  ( Yes ?, No ?, are equal ? )

NOTE: All tasks from this laboratory, should be implemented using <stdio.h> library for all user interface input/output communication.


TASK L2.T3

Description of the task:

Write a program that reads 4 integers from the keyboard
and then checks how many repetitions (of the same values) are among them:
  • "doubles":  (2,5,3,10) , (2,7,1,7) , (2,2,5,5) , . . .
  • "threes": (6,3,6,6) , . . .
  •  or "fours": (5,5,5,5)
The checking process should use rules analogous to card games like Poker.
