TASK L1.T1

Description of the task to be performed:

Write a program which reads the length (a,b,c) of three sides of the triangle, and later calculates:

a) the perimeter of this triangle         // ( a+b+c )

b) and the area of this triangle (check Heron's formula:  https://en.wikipedia.org/wiki/Heron%27s_formula)


TASK L1.T2

Description of the task:

Write a program that calculates mathematical expressions defining:
a) surface area and circumference of a circle with the radius R, (the value of R should be entered by user)
b) the distance between two geometric points, whose Cartesian coordinates
   are defined by four real numbers (x1, y1) and (x2, y2), also entered by user.
   (hint: https://en.wikipedia.org/wiki/Euclidean_distance)

TASK L1.T3

Description of the task:

Write a program which classifies the height of people.
The program should ask the user, about the height of a person (in centimeters)
and later display one of the three classification labels:  Low  < 150cm <  Average  < 180cm <  High
