TASK L7.T1

Description of the task to be performed:

Write a new "super" program, which will combine into one file, selected programs (tasks solutions) created during previous laboratories. Please select 2 tasks/sub-programs from laboratory 3, two from laboratory 4, and two from laboratory 5, or any similar combination.

In the simplest version, all previous functions
     int main()

could be converted to  parameterless sub-functions, eg.
     void DisplayMyBusinessCard( void )
     void ClassifyPeopleHeight( void )
     void ClassifyPressedKeys( void )
     void PrintChristmasTree( void )
   . . .  etc.

New function main(), in this "super" program, can sequentially call all the above sub-functions.
It can look similary to the example program, from second/middle column, on the slide/page number 2 (from lecture 5): L05_functions.pdf

Hint: to concatenate the code of existing programs, you can use commands operating on a editor's clipboard ( Cut / Copy / Paste )

TASK L7.T2

Description of the task:

Expand the program (Lab7_Task1) by adding the "Menu", which allows to control the sequence of executed/called functions and exit the program. Sample menu may have options:

    Menu:                                                  
          1. Display my business card         
          2. Classify the height of person    
          3. Print Christmas Tree                
          4. Exit                                           
    Select the number of operation: ... ? 

Using this menu, it should be possible to execute the functions in any order (decided during execution runtime).

TASK *L7.EXTRA (optional)

Description of the task:

Advanced students,may write a more difficult version of "MENU",
implemented in the form of separate function with arguments:
    int Menu( char* menu_options[], int number_of_options, int default_option);   
returning the int index of option selected by pressing the "ENTER" keyboard button.

The choice of options could be made by using the "ARROW" keys, to move the options pointer / backlight.
