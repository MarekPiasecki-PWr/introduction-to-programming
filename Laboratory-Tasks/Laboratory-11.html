TASK L11.T1

Description of the task to be performed:

Using the struct type to organise (group/agregate) nontrivial data collections.

Declare the struct ArrayWithCounter, which represents / contains:
   - numbers - 10 elements array of double numbers,
   - size - an integer counter, informing how many (from above 10) elements contain valid data (logical size).
In C++, the struct declaration, may even contain the definition of maximal (physical) size
   - static const int MAX_SIZE = 10;

Declare the struct StatResults, which represents / contains three fields:
    -  avg - the arithmetic average value (calculated for specified array)
    -  min - the minimal value
    -  max - the maximal value

Write a function CalculateStatistics which analyse the content of an array (received as an argument), calculate basic statistics (avg, min, max) and return calculated results in the form of struct:
      StatResults  CalculateStatistics(            ArrayWithCounter   inputArray );         // "by value",  (huge copy) ,  possible in C and C++
or alternatively:
      StatResults  CalculateStatistics( const ArrayWithCounter &inputArray );       // "by reference", possible in C++ only
or
      StatResults  CalculateStatistics( const ArrayWithCounter *inputArray );        // "by pointer", possible in C and C++
 

Illustration, how such function could be defined, can be found at the bottom the page 3 in Lecture_09 SLIDES
or in TutorialsPoint tutorial:  "Structures and Functions in C"
  

Example demonstration / testing program / for expected target function CalculateStatistics:

     ArrayWithCounter  arr,  second_array,  array_3;    
     StatResults  result;                                                 
     . . .                                                                            
     arr.size=2;                                                               
     arr.numbers[0]=123.4;                                            
     arr.numbers[1]=567.8;                                            
     . . .                                                                            
     result = CalculateStatistics( arr );                           
     cout << "The average = " << result.avg;
